# mini-sql-db

A file-based SQL database written in JavaScript. Uses SQLite-like syntax for querying and managing data.

## Folder Structure

mini-sql-db/
|--- lib/
|    |---
|--- logger/
|    |--- logger.js
|--- index.js
|--- package.json
|--- README.md
|--- sql_syntax.txt

## Setup

- Clone the project
- Run `npm install` inside the project
- Create a `.env` file at root
